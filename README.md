# AI-Driven 5G Network Traffic Prediction

## Overview
This notebook demonstrates the application of machine learning models to analyze and predict 5G network traffic metrics using the IW-IB-5GNET dataset. It focuses on modeling virtual network function (VNF) deployment and performance prediction through data preprocessing, exploratory analysis, and training of Random Forest and LSTM models.

## Features
- Data cleaning, normalization, and feature selection for 5G network data.
- Visualizations of network traffic metrics and rule complexity.
- Implementation and evaluation of a Random Forest regression model.
- Preparation of sequential data and training of an LSTM neural network for time-series prediction.
- Comparative performance analysis of Random Forest and LSTM models.

## Tech Stack
- Python
- Jupyter Notebook / Google Colab
- pandas, numpy
- seaborn, matplotlib
- scikit-learn
- TensorFlow / Keras
- missingno

## How to Use
1. Clone the repository.
2. Upload or link the IW-IB-5GNET dataset CSV file (`iwib5gnet_v0.csv`).
3. Open the notebook `ai-driven-5g-network-traffic-prediction.ipynb` in Google Colab or Jupyter.
4. Run the notebook cells sequentially to preprocess data, explore features, train models, and evaluate performance.
